Latest Project

SMC Course Registration bot

Utilizes gmail API to scan for emails, and then utilizes Selenium to navigate the webpage and automate browser operations. Regex is used to determine from the JSON objects collected from GMail API to decide if a course is open. For selecting elements on the page, robust selectors were used to ensure that going into the future, the bot would still work even if minor adjustments to id were made. Has both a single register and bulk register mode.

Technologies Used:

  • HTML/CSS
  • GMail API, PubSub
  • Python
  • Selenium
